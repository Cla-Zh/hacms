## 3. 风格与约束 (保持不变)

### 3.1 UI 风格融合
- **Bloomberg**: 命令栏 + ticker + 数据密度
- **TradingView**: 热力图 + 行业轮动
- **Apple**: 留白 + 阴影 + 圆角
- **Linear**: 浅色 + 干净 + 信息层次

### 3.2 当前实现
- **配色**: hacms 浅色主题 (跟 sidebar 标签一致, 去 AI 味)
  - 主背景 `#fafbfc`
  - 卡片 `#ffffff` + `#f8fafc` 渐变
  - 主色 `#3b82f6` (蓝) / `#10b981` (绿涨) / `#ef4444` (红跌) / `#ffa500` (橙警示)
  - 文字 `#1e293b` / `#475569` / `#64748b` / `#94a3b8`

### 3.3 数据要求 (本次新增约束)
- **强制**: 所有数据必须带 `source_name` + `source_url` + `fetch_timestamp_utc`
- **禁止**: 社交媒体 / 小道消息 / 分析师博客作为数据来源
- **允许**: 官方 (政府/央行/财政部/统计局) / 交易所 / 上市公司披露 / 权威指数公司 / 监管机构
- **争议数据**: 加备注 + 双 source (e.g. S&P Global + Bloomberg)
