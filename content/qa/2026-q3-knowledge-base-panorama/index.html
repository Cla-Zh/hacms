💡 Executive Summary · 全景洞察

AI 知识库正在从 'Naive RAG' 向 'Agentic RAG' 跃迁, 检索层从 '单一向量' 向 '混合+图+数据库内 AI' 演进, 安全层从 '单点 Prompt 防御' 升级为 '端到端 Security Stack'。三大趋势: (1) Agentic RAG 成为新范式 — Modular RAG (Gao 2024) / GraphRAG (Microsoft 2024) / Self-RAG (Asai 2023) / CRAG (Yan 2024) 已被 OpenAI Responses (2025-03)、Anthropic MCP (2024-11 发布, 2025-12-09 加入 Linux Foundation AAIF, CRITICAL 事件)、AWS Bedrock KB Agentic (2025)、Vertex RAG Engine (2024-12) 落地, LangChain / LlamaIndex / DSPy / Letta 提供框架支持; (2) 检索基础设施 GPU 化 + 库内 AI 化 — NVIDIA cuVS (CAGRA/IVF-PQ, 2024-09 GA)、AlloyDB AI、Aurora ML、pgvector、Oracle 23ai (2024-05 GA) 把向量检索从独立服务收编回数据库主战场, DiskANN/SPANN/HNSW 让万亿级向量具备工程可行性; (3) Build-time Security 主流化 — OWASP Top 10 for LLM 2025、NIST AI RMF 1.0、MITRE ATLAS、Anthropic RSP v3.3、EU AI Act (5 阶段 2024-2027) 共同确立标准, NeMo Guardrails / Guardrails AI / Lakera Guard / Prompt Armor 形成产品矩阵, VC 在 Guardrails 方向下注明确。商业上, 全球 RAG 市场 MarketsandMarkets 2025-10 预测 2024-2030 CAGR ~38.4%, 2030 达 ~9.86B USD (2024=$1.36B, 2025=$1.94B); 头部融资: Pinecone B 轮 100M (2023-04) 估值 750M / Glean E 轮 260M 估值 4.6B (2024-09) / Hebbia B 轮 130M 估值 700M (2024-07) / Cognition/Devin A 轮扩展 175M (2024-04, 现估值 26B 2026-05) / Manus $75M Series A (2025-04, 估值 500M).

块 1 · AI 知识库架构全景 (Knowledge Base Architecture)

📋 块摘要

RAG 自 2020 年提出以来, 经历了 Naive RAG → Advanced RAG → Modular RAG → Graph RAG → Agentic RAG 五代演进。每代演进都对应明确的论文锚点和大公司落地, 现阶段已进入 'Modular + Agentic' 双轨成熟期。

🚀 5 代演进路径

2020
Naive RAG
Retrieve-then-read, 单一向量检索 + 单次 LLM 生成
📄 Lewis et al. 2020 (arXiv:2005.11401)
2022
Advanced RAG
Pre-retrieval 优化 (query rewrite) + Post-retrieval 优化 (rerank)
📄 Izacard et al. Atlas 2022 (arXiv:2208.03299)
2024
Modular RAG
模块化重组, 检索/重排/生成/校验可插拔
📄 Gao et al. 2024 (arXiv:2407.16833)
2024
Graph RAG
图谱增强, 社区检测 + 跨实体推理, 解决 multi-hop 难题
📄 Microsoft GraphRAG 2024 (Edge et al.)
2023
Self-RAG / CRAG
模型自反思检索质量 + 校正器
📄 Asai 2023 Self-RAG (arXiv:2310.11511) + Yan 2024 CRAG (arXiv:2401.15884)
2024
Agentic RAG
Agent 自主决定检索时机、工具选择、多步推理
📄 OpenAI Responses 2025-03 + Anthropic MCP 2024-11

块 2 · 数据库检索全景 (Database Retrieval)

📋 块摘要

向量数据库在 2024-2025 进入'成熟+分化'阶段: ANN 算法 (HNSW/IVF-PQ/DiskANN/SPANN/ScaNN/CAGRA) 各有适用场景, 独立 Vector DB (Pinecone/Weaviate/Qdrant/Milvus/Chroma) 与'数据库内 AI' (AlloyDB AI/Aurora ML/pgvector/Oracle 23ai) 并行, GPU 加速 (NVIDIA cuVS) 成为标准。
