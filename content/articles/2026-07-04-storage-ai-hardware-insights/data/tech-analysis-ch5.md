## §5.1 三大趋势 — AI 一体机 / CXL 池化 / 国产替代

### 趋势 1: AI 存储一体机进入主流

2025 全球 AI 一体机出货 ~ 8 万台 (DGX SuperPOD / 华为 Atlas 900 / 浪潮元脑等), 2026 预计 18 万台, 同比 +125% [来源: TrendForce + IDC 综合估算]。

**典型形态**:
- **国际**: NVIDIA DGX SuperPOD (GB200 NVL72 + BlueField-3 + Quantum-2) 单柜 72 卡 + 30TB HBM3E + 1.5PB NVMe
- **国产**: 华为 Atlas 900 (Ascend 910C + OceanStor Dorado 300K) 单柜 64-256 卡
- **独角兽**: 浪潮 PSOAR 浸没式液冷 + 昇腾 910C, 单柜 80kW+ 功率密度

**关键技术**:
- **GPUDirect Storage 1.4**: 绕过 CPU 直达 NVMe, 延迟降 70%
- **BlueField-3 卸载 NVMe-oF target**: 主机 CPU 释放 30%+
- **CXL Memory 扩 KV cache**: 显存 5x 等效, 延迟 +30%

代表性论文 **AI Storage Survey ACM CSUR 2024** 与 **CXL fabric ISCA 2024** 系统梳理了一体机架构关键创新 [来源: 论文 DOI]。

### 趋势 2: CXL 内存池化 2026 H2 首批大规模部署

**里程碑预测**:
- 2026 H1: 三星 / 海力士 / 美光 CXL 2.0 256GB 模块量产 [来源: https://news.samsung.com/global/samsung-electronics-starts-mass-production-of-industry-first-cxl-memory-module]
- 2026 H2: 首批 CXL 内存池化数据中心部署 (微软 Azure / AWS / 阿里云)
- 2027 Q1: CXL 3.0 fabric 首批服务器

**应用场景**:
- AI 推理: KV cache 卸载到 CXL, 单卡支持更大模型
- 大数据分析: Spark on CXL, 减少 DRAM 成本 50%+
- 内存数据库: Redis / Memcached CXL 扩展, 单实例 10TB

**关键厂商**:
- 国际: Samsung / SK hynix / Micron / Memverge (DaP 5.0)
- 国产: 华为 OceanStor CXL / 紫光得瑞 CXL-MM / 浪潮 PSOAR CXL

代表性论文 **CXL fabric ISCA 2024** + **CXL switch design NSDI 2024** 给出 fabric 架构的关键设计 [来源: 论文 DOI]。

### 趋势 3: 国产替代 DPU + PMem 双线推进

**DPU 国产替代**:
- 2026 国产 DPU 市场份额目标 30% (vs 2024 ~ 12%)
- 主力厂商: 中科驭数 K2 / 芯启源 TC8210 / 益思芯 / 云豹智能
- 信创集采: 中国移动 / 电信 / 联通 2024 集采名录已纳入

**PMem 国产替代**:
- 傲腾退役后, CXL PMem 国产化窗口期 2024-2028
- 长江存储 / 中科院微电子所联合 Xtacking SCM 2026 Q3 量产
- 紫光得瑞 CXL-MM 256GB 2025 试产

**国产替代政策驱动**:
- 信创目录 2024 扩容至 DPU / CXL 设备
- 关基条例 + 数据安全法执法趋严
- 国家集成电路产业投资基金 (大基金三期 2024.5 成立 CNY 3440 亿) 加码

## §5.2 CISO / 架构师 / CFO 行动清单

### CISO (首席信息安全官) — 4 行动

#### 行动 1: 硬件 TEE 试点
- **目标**: 2026 H1 选定 1-2 个 AI 训练集群试运行 TDX / SEV-SNP / 昇腾 NPU TEE
- **指标**: 关键数据 (用户隐私 / 训练数据) TEE 覆盖度从 0% 提到 30%+
- **预算**: 50-200 万 CNY (含软件适配 + 性能调优)
- **参考**: Intel TDX 1.0 / AMD SEV-SNP 在 EPYC Genoa / NVIDIA H100 CC (GPU TEE H100 CCS 2024)

#### 行动 2: 加密卸载到 DPU
- **目标**: 把 IPSec / TLS 卸载到 BlueField-3 / 国产 DPU, 主机 CPU 释放 30%+
- **指标**: HTTPS 加密流量主机 CPU 占用从 35% 降到 5% 以下
- **预算**: 100-300 万 CNY (DPU 采购 + 部署)
- **参考**: RDMA encryption NDSS 2024 (线速 100GbE IPSec < 5% 开销)

#### 行动 3: 零信任架构落地
- **目标**: 把存储 / 计算 / 网络三层纳入零信任 (BeyondCorp 模式)
- **指标**: 关键系统 RBAC 100% 覆盖, 默认拒绝策略
- **预算**: 200-500 万 CNY (IAM + 策略引擎 + 培训)
- **参考**: NIST SP 800-207 (零信任架构)

#### 行动 4: 国产替代合规
- **目标**: 关基行业 (金融/政务/能源) 数据本地化 + 国产硬件采购 50%+
- **指标**: 信创目录覆盖率从 ~ 30% 提到 60%+
- **预算**: 1000 万 - 1 亿 CNY (按企业规模)
- **参考**: 信创目录 2024 扩容清单

### 架构师 — 4 行动

#### 行动 1: CXL 内存试点
- **目标**: 2026 Q2 选定 1 个推理集群试运行 CXL 1.1/2.0 Memory Module
- **指标**: KV cache 显存扩展 3-5x, 单卡推理容量提升
- **预算**: 200-500 万 CNY (CXL-MM + 服务器升级)
- **参考**: CXL Survey TACO 2024 / Memverge DaP 5.0 / 华为 OceanStor CXL

#### 行动 2: SmartSSD 计算存储
- **目标**: 2026 Q3 把向量数据库 / RAG 推理下推 ScaleFlux CSD 3320 / 忆芯 STAR 1500
- **指标**: 向量检索 QPS 提升 3-5x, CPU 释放 50%
- **预算**: 100-300 万 CNY (SmartSSD 替换通用 SSD)
- **参考**: POLYNIS SIGMOD 2023 (12 种下推算子) / NDP++ HPCA 2024 (Transformer 2.1x)

#### 行动 3: DPU 卸载网络/存储
- **目标**: 2026 H1 在 10% 集群部署 BlueField-3 / 中科驭数 K2 卸载 NVMe-oF target + 压缩
- **指标**: 主机 CPU 占用降到 < 30%, 单集群吞吐 +50%
- **预算**: 300-800 万 CNY (DPU + 网卡 + 软件)
- **参考**: Mellanox ASAP2 OSDI 2023 / Pensando DSC OSDI 2023

#### 行动 4: AI KV 缓存分层
- **目标**: 把 LLM KV cache 分层 (HBM 热 + CXL 温 + QLC 冷), 显存利用率 +200%
- **指标**: 单卡支持上下文长度从 32K 提到 128K, batch size +2x
- **预算**: 100-400 万 CNY (CXL-MM + SSD 扩容)
- **参考**: LLM KV cache offload ASPLOS 2024 / vLLM paged attention SOSP 2023 / KV cache offload Survey TACO 2024

## §5.3 风险预警 — 出口管制 / 生态锁定 / 标准分裂

### 风险 1: BIS 规则再升级 (高概率)

**触发条件**:
- 2025 Q4 - 2026 Q2 美国对华 AI 芯片规则可能再加码
- 阈值可能从 FP64 + I/O 进一步收紧到 FP8 / FP16 + HBM 容量
- H20 / L40S / L4 都可能纳入新一轮禁运

**触发事件**:
- 2024.12 BIS 规则已将 H20 纳入 [来源: BIS 公告 2024.12]
- 2025.01 可能再次扩展到 L40S / L4 / 国内特供 RTX 系列
- 2026 H1 可能对国产昇腾 910C 实施"外国直接产品规则" (FDPR)

**应对策略**:
- 加快国产替代 (昇腾 910C / 寒武纪 590 / 海光 Z100) 部署
- 设立 BIS 风险准备金 (CFO 行动 4)
- 多区域备份 (新加坡 / 中东 / 拉美) 数据中心

### 风险 2: 国际大厂生态锁定 (中概率)

**NVIDIA CUDA 13 绑定路径**:
- 2026 CUDA 13 进一步绑定 BlueField-3 + Grace CPU + Spectrum-X
- 第三代 NVLink 带宽 900 GB/s 锁定整机柜
- 一旦使用, 5 年内难以替换

**AMD ROCm 7 绑定路径**:
- 2026 ROCm 7 绑定 Pensando DPU + MI400 + EPYC Turin
- 试图打破 CUDA 垄断, 但生态成熟度差距 3-5 年

**应对策略**:
- 双源采购 (NVIDIA + AMD + 国产) 避免单一供应商
- 国产 + 国际 7:3 配比 (关基行业可调至 9:1)

### 风险 3: 国产替代节奏与新版规则赛跑 (中概率)

**国产替代现状**:
- 2026 H1 国产 AI 卡在性能上接近 NVIDIA H20 (算力对标)
- 但在 CUDA 生态成熟度 / 开发者群体 / 软件栈上仍有差距
- 国产 NAND / DRAM 在良率 / 产能 / 单位 $/GB 上仍落后国际

**风险点**:
- 国产替代节奏 (2026-2028) 可能被 BIS 规则再升级打断
- 大基金三期 CNY 3440 亿 资金到位节奏 (2024-2027) 影响产能扩张

**应对策略**:
- 分阶段替代: 推理先行 (对标 H20) → 训练后行 (对标 H100/H200)
- 国产 + 国际并行, 关键业务 (核心训练) 留国际备份

### 风险 4: 标准分裂 (低概率但影响大)

**CXL vs Gen-Z + OpenCAPI**:
- CXL 主导 (Intel/三星/海力士/美光/ARM 阵营)
- Gen-Z 已基本退出竞争
- 但 CXL 在 2026 仍面临 UCIe / UCIe 3D 等新标准挑战

**NVMe-oF vs NVMe-oF/TCP 内部路线分裂**:
- RoCE v2 vs TCP 两种传输在不同云厂商之间分裂
- 未来可能形成 NVMe/RoCE 联盟 vs NVMe/TCP 联盟

**应对策略**:
- 优先采用 OCP/OCP Cloud SSD v2 等事实标准
- 单一标准押注风险大, 多标准并行测试

代表性论文 GPU TEE H100 CCS 2024 与 RDMA encryption NDSS 2024 提供了 GPU + 网络加密的最新基准 [来源: 论文 DOI]。
