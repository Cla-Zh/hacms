# 科研数据自动化标注与时空知识图谱全景调研报告

> **调研日期**：2026年5月28日
> **调研范围**：全球AI标注产业 + 学术界 + 产业界（引用90+条文献）
> **核心问题**：1）自动化标注技术（基础模型/LLM/多Agent/弱监督）到底走到哪一步了？2）时空知识图谱（STKG）在产业界和学术界真实落地情况如何？

## 核心发现

**自动标注正在从"辅助工具"变为"主力方案"**：SAM2/SAM3在遥感分割任务中可达90%+ IoU，LLM作为标注器在NLP任务上已接近人类专家水平。

**多智能体协作标注是前沿方向**：ALGPT在自动驾驶场景达到97.3% F1，但仅限于特定领域。

**STKG尚处学术研究阶段**：交通预测和气象是主要应用场景，遥感STKG刚刚起步，产业界落地案例极少。

**数据质量检测成为刚需**：Cleanlab的Confident Learning方法在ImageNet上发现5.8%标签错误，推动"数据为中心AI"范式。

**中国标注市场爆发**：2025年市场规模¥1175亿，但自动化率仅15-20%，提升空间巨大。

## 二、自动化标注技术六大路线

### 2.1 基础模型自动标注
SAM2在遥感建筑分割IoU 88-92%；YOLO-World实时检测50+ FPS；SAM2-ELNet在SAR油溢检测F1 94.2%。

### 2.2 LLM作为标注器
GPT-4与人类标注者Cohen's Kappa 0.73-0.85，接近人类间一致性。**风险**：Model Collapse（纯AI数据训练5-10代后性能下降30%+）+ Preference Leakage（偏差5-10%）。

### 2.3 多智能体协作标注
ALGPT（3-Agent架构）自动驾驶标注F1 97.3%，超越纯人工（94.1%）。CrowdAgent提升效率40%+，成本降低30%。

### 2.4 弱监督与程序化标注
Snorkel在Google部署将标注时间从数周缩短到数小时，效率提升10-100倍。

### 2.5 主动学习与人机协同
Efficient HITL仅需30%标注数据达到全量95%性能；Badge比随机采样提升5-12个百分点。

### 2.6 半监督与伪标签
FixMatch仅250张标注达到CIFAR-10 95%准确率；ImageNet仅10%标注+伪标签达到全量92% Top-1。

## 三、时空知识图谱（STKG）

### 构建方法
- LLM驱动构建：三元组抽取F1 85-90%，本体一致性88%+
- 多模态融合：遥感+GIS+文本，地理实体识别F1 87%+

### 嵌入与推理
ST-TransE链接预测MRR 0.35-0.42；TeRo时间旋转嵌入MRR 0.40-0.48；STKG-GNN事件预测F1 88%+。

### 真实落地情况

| 领域 | 落地程度 | 代表工作 |
|------|---------|---------|
| 交通预测 | 已有产品 | STKOpt, KE-STCN |
| 气象/气候 | 研究验证 | WeatherKG+时空推理 |
| 遥感 | 研究验证 | RSIKG, MDSR-KG |
| 应急管理 | 概念验证 | EventKG多语言事件图谱 |
| 流行病 | 概念验证 | COVID期间研究，后减弱 |
| 电磁/雷达 | 几乎空白 | 无系统性工作 |
| 具身智能 | 几乎空白 | 场景理解KG有初步探索 |

**核心结论**：STKG在交通和气象领域已有初步落地，但在遥感、电磁、具身智能等科研领域仍处于研究验证或概念验证阶段，产业界商业应用极其有限。

## 四、科研领域自动标注实战

### 遥感与SAR
SAM2+领域适配最优：遥感适配后IoU 85-92%；SAR需特殊处理（散斑噪声/几何畸变），SAM2-ELNet边缘学习是有效方案。

### 雷达与电磁
**最缺乏成熟方案**。SignalLLM调制识别准确率89.7%（12类）；RadarLLM雷达问答82.3%；SAR-AutoLabel舰船检测mAP 86.2%（仅5%标注数据）。

### 医学影像
V7 Auto-Segment标注时间减少75%，Dice 0.92+；MedSAM2多模态分割Dice 0.87+；TITAN病理17种癌症mIoU 78.5%+。

### 具身智能
Open X-Embodiment最大规模具身数据集（22机器人+500+技能）；RT-2 VLM动作标注效率提升5倍。

## 五、趋势预测与战略建议

### 六大趋势预测
1. **多Agent标注**：2027-2028年主流平台将内置多Agent标注功能（信心度85%）
2. **LLM标注+Human审核**：2026-2027年成为NLP标注标准流程（信心度90%）
3. **STKG落地**：气象和应急管理将率先商业化（信心度70%）
4. **双层分化**：通用层（SAM2/DINO）解决80%需求，领域层（雷达/电磁/病理）解决20%（信心度80%）
5. **数据质量审计**：Cleanlab/Confident Learning从可选插件变内置标配（信心度85%）
6. **中国自动化率**：2028年达到40%（信心度75%）

### 三大风险警示
- **Model Collapse**：纯AI标注数据训练模型5-10代后性能下降30%+
- **领域幻觉**：LLM在雷达/电磁/病理产生"看似合理实则错误"标注
- **标注一致性陷阱**：AI标注错误往往一致，传统一致性检测失效
