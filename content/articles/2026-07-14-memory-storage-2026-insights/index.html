8.1 Agent 4 质控发现 — Vector DB 基础层被忽视

⚠️ 重要遗漏: Pinecone / Weaviate / Milvus / Qdrant / Chroma 等核心 vector database 是 Agent Memory 的底层基础设施. Mem0 已整合 20 个 vector store 后端, 但 3 个执行组报告完全未把 vector DB 当独立品类列出. 补遗: 这些 vector DB 多数 2025-2026 都加了 native agent memory API (Pinecone assistant / Weaviate generative search).

第十三章 · 结论 + 风险 + 选型决策树

13.1 核心结论

  1. 品类已确立: 2025-10 Mem0 $24M + 2025-11 Microsoft Foundry + 2026-02 Vertex GA + 2026-04 Google Agent Identity = "Memory as a Service" 成形, 不可逆.
  2. 4 大学派并存: Vector+KV (Mem0) / 时序图谱 (Zep) / OS 分层 (Letta) / 多网络 Belief (Hindsight) — 没有银弹, 按场景选型.
  3. 评估体系 3 维: 检索精度 (LongMemEval / LoCoMo / BEAM) + 延迟 (p95 < 300ms) + 成本 (tokens/query), 厂商自报 vs 独立评估差距巨大, 务必跑独立 benchmark.
  4. 学术界追工业界: Hindsight (4 网络) / Memori (语义三元组) / Mooncake (KVCache-centric) 已经在生产可用, 学术界 2026 在追赶.
  5. 中国生态滞后但加速: MemTensor / MemOS / OPPO 小布 是中国"重 RAG 轻 Memory" 趋势下的反击, 但跟美国仍有 6-12 个月差距.
